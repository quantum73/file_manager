File manager

File manager for 3D models

Getting Started

Installation

  • Clone this repo: 

    git clone https://github.com/quantum73/file_manager.git
cd ../file_manager

  • After cloning the repository you need to add two folders (data and img) in root folder ../file_manager

  • For pip users, please type the command pip install -r requirements.txt.

Run server

python main.py
  • Copy this address and paste it into the browser's address bar

Manual

Кнопка Home

Кнопка Home возвращает в корень

Поле создания новой папки (тэга)

В поле New folder вводится название папки (тэга), которую вы ходите создать.
Нажатие кнопки Create создаёт данную папку.

Поле загрузки файла модели

Выбирается нужный файл модели для загрузки на сервер, нажатие кнопки Upload загружает файл на сервер.

Примечания: - За раз можно загрузить только один файл - Поддерживаемые расширения файлов: .obj, .zip, .rar, .blend, .fbx - Если формат не соответствует ни одному из вышеперечисленных, выводится сообщение Wrong format file:

Поиск по тэгу/описанию

В поле поиска Search Tag/Description вводится название тэга или часть описания, в зависимости от того, что вы
хотите найти. Кнопка Tag выполняет поиск по тэгу(название папки), а кнопка Description выполняет поиск
по описанию, соответственно.

  • Если тэг не был найден, то выводится следующее сообщение:

Добавление модели в виде ссылки на Buffer

Чтобы добавить модель в виде ссылки на Buffer введите в поле Buffer link ссылку на файл в Buffer и нажмите Add link
